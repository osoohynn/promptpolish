PromptPolish 개인정보처리방침

PromptPolish는 사용자의 개인정보 보호를 최우선으로 생각합니다. 본 개인정보처리방침은 확장 프로그램이 어떤 데이터를 수집하고 사용하는지 설명합니다.

1. 수집하는 데이터

PromptPolish는 다음과 같은 데이터만 수집합니다:

  • 사용자 입력 텍스트: 프롬프트 개선을 위해 사용자가 입력한 텍스트만 처리합니다.
  • AI 플랫폼 선택 정보: 사용자가 선택한 AI 플랫폼(ChatGPT, Claude, Gemini, Perplexity) 정보는 브라우저 로컬 저장소에만 저장됩니다.

수집하지 않는 정보:

  • 개인 식별 정보 (이름, 이메일, 전화번호 등)
  • 웹 브라우징 기록
  • 위치 정보
  • 인증 정보 (비밀번호, API 키 등)

2. 데이터 사용 방식

수집된 데이터는 다음 목적으로만 사용됩니다:

  • 프롬프트 개선: 사용자가 입력한 텍스트를 OpenAI API를 통해 더 명확하고 구조화된 프롬프트로 개선합니다.
  • 설정 저장: 사용자가 선택한 AI 플랫폼 정보를 로컬 저장소에 보관하여 다음 사용 시 자동으로 적용합니다.

3. 제3자 공유

PromptPolish는 프롬프트 개선을 위해 다음 제3자 서비스를 사용합니다:

  • OpenAI API: 사용자가 프롬프트 개선을 요청할 때, 입력한 텍스트가 OpenAI의 GPT-4o-mini 모델로 전송됩니다.
    OpenAI 개인정보처리방침: https://openai.com/privacy
  • Cloudflare Workers: 백엔드 서버로 사용되며, OpenAI API 요청을 중계합니다. 사용자 데이터를 저장하지 않습니다.
    Cloudflare 개인정보처리방침: https://www.cloudflare.com/privacypolicy

중요: PromptPolish는 사용자 데이터를 판매하거나, 광고 목적으로 사용하거나, 제3자에게 제공하지 않습니다. 데이터는 프롬프트 개선 목적으로만 전송되며, 서버에 저장되지 않습니다.

4. 데이터 보관

  • 로컬 저장소: AI 플랫폼 선택 정보만 브라우저 로컬 저장소(chrome.storage.local)에 저장됩니다. 이 데이터는 사용자 기기에만 존재하며 외부로 전송되지 않습니다.
  • 서버 저장: PromptPolish 백엔드는 사용자 입력 텍스트나 개선된 프롬프트를 저장하지 않습니다. 모든 데이터는 요청 처리 즉시 삭제됩니다.

5. 데이터 보안

  • 모든 데이터 전송은 HTTPS를 통해 암호화됩니다.
  • OpenAI API 키는 사용자에게 노출되지 않으며, 백엔드 서버(Cloudflare Workers)에서만 관리됩니다.
  • 사용자는 언제든지 확장 프로그램을 삭제하여 로컬 저장소 데이터를 제거할 수 있습니다.

6. 사용자 권리

사용자는 다음과 같은 권리를 가집니다:

  • 데이터 삭제: 확장 프로그램을 삭제하면 로컬 저장소의 모든 데이터가 자동으로 삭제됩니다.
  • 서비스 중단: 언제든지 확장 프로그램 사용을 중단할 수 있습니다.

7. 아동 개인정보 보호

PromptPolish는 만 13세 미만 아동으로부터 고의로 개인정보를 수집하지 않습니다.

8. 개인정보처리방침 변경

본 개인정보처리방침은 필요에 따라 업데이트될 수 있습니다. 중요한 변경 사항이 있을 경우 확장 프로그램을 통해 공지합니다.

PromptPolish Privacy Policy

PromptPolish prioritizes user privacy. This Privacy Policy explains what data the extension collects and how it is used.

1. Data We Collect

PromptPolish only collects the following data:

  • User Input Text: Only the text you enter to improve your prompts is processed.
  • AI Platform Selection: Your selected AI platform (ChatGPT, Claude, Gemini, Perplexity) is stored in your browser's local storage only.

We do NOT collect:

  • Personal identifiable information (name, email, phone number, etc.)
  • Web browsing history
  • Location data
  • Authentication credentials (passwords, API keys, etc.)

2. How We Use Your Data

Collected data is only used for:

  • Prompt Improvement: Your input text is sent to OpenAI API to generate clearer and more structured prompts.
  • Settings Storage: Your AI platform preference is saved locally for future use.

3. Third-Party Sharing

PromptPolish uses the following third-party services for prompt improvement:

  • OpenAI API: When you request prompt improvement, your input text is sent to OpenAI's GPT-4o-mini model.
    OpenAI Privacy Policy: https://openai.com/privacy
  • Cloudflare Workers: Used as our backend server to relay API requests. No user data is stored.
    Cloudflare Privacy Policy: https://www.cloudflare.com/privacypolicy

Important: PromptPolish does NOT sell user data, use it for advertising, or share it with third parties. Data is only transmitted for prompt improvement and is not stored on our servers.

4. Data Storage

  • Local Storage: Only AI platform selection is stored in your browser's local storage (chrome.storage.local). This data remains on your device and is never transmitted externally.
  • Server Storage: PromptPolish backend does NOT store user input text or improved prompts. All data is deleted immediately after processing.

5. Data Security

  • All data transmission is encrypted via HTTPS.
  • OpenAI API keys are never exposed to users and are only managed on our backend server (Cloudflare Workers).
  • You can delete all local storage data at any time by uninstalling the extension.

6. Your Rights

You have the following rights:

  • Data Deletion: Uninstalling the extension automatically deletes all local storage data.
  • Service Termination: You can stop using the extension at any time.

7. Children's Privacy

PromptPolish does not knowingly collect personal information from children under 13 years of age.

8. Privacy Policy Changes

This Privacy Policy may be updated as needed. Significant changes will be announced through the extension.
